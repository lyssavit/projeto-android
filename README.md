# projeto-android

Projeto desenvolvido durante o curso de HTML5 e CSS3 do Curso em Vídeo, ministrado por Gustavo Guanabara.
O site apresenta a história e evolução do sistema Android, utilizando um layout moderno e responsivo. Neste projeto foram aplicados conceitos de HTML5 semântico, CSS3, responsividade, tipografia, cores, vídeos incorporados e boas práticas de desenvolvimento web.
